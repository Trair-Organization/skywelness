# UI Stil Kuralları

## Arka Plan Rengi
- Tüm bileşenlerde (modal, input, buton, kart) arka plan rengi **beyaz (#ffffff)** olmalı.
- Koyu tema (`rgba(15,23,42,...)`, `#0f172a` arka planlar) kullanma — her şey beyaz/açık tema.
- Aktif butonlar: `background: #eff6ff` (açık mavi), `color: #2563eb`
- Pasif butonlar: `background: #ffffff`, `border: 1px solid #e2e8f0`, `color: #374151`
- Input'lar: `background: #ffffff`, `border: 1px solid #e2e8f0`
- Modal'lar: `background: #ffffff`, `border: 1px solid #e2e8f0`

## Genel Kurallar
- Metin renkleri: Primary `#0f172a`, Secondary `#374151`, Muted `#64748b`
- Border rengi: `#e2e8f0`
- Accent: `#2563eb` (mavi)
- Success: `#059669` (yeşil)
- Danger: `#dc2626` (kırmızı)
- Warning: `#d97706` (turuncu)
